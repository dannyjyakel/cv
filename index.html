Next are tables

Tables contain the "Table" entry, and then within that entry you have:

  • "tr" or the table row
  • "td" or table data which is the cell going across the "tr"
  • "th" or table header which will be the header of each column

But, to help with styling (once CSS is learned), place the "th" and "td" inside the "thead" and "tbody"

Example:

Header 1 Header 2
Row 1, cell 1 Row 1, cell 2
Row 2, cell 1 Row 2, cell 1
